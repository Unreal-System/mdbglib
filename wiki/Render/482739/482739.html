mdbglib is a managed library for interacting with Debugger Engine API (dbgeng.dll), which is supplied by Microsoft as part of Debugging Tools for Windows.

mdbglib will help you if you need to:
  1. Learn how to use Debugger Engine API from managed code (mostly C#)
  2. Extract information from a dump
  3. Extract information from the memory of a running application
  4. Modify virtual memory of a running application
  5. Write Powershell scripts to interact with a Debugger engine
  6. (possibly somewhere in the future) write your own debugger.

mdbglib provides both a thin and a thick wrapper around Debugger Engine API.
  1. The thin wrapper gives access to Debugger Engine API from managed code doing the required type conversions. It is easier to use for people familiar with Debugger Engine API or for people who want to achieve some custom, advanced behavior
  2. The thick wrapper tends to provide more .NET like, object oriented, experience in working with Debugger Engine API. It hides calls to the undelying API and provides easy to use objects such as DbgThread, DbgFrame, DbgModule, DbgObject, DbgType, ... Think about the think wrapper compared to dbgeng API as of Windows Forms compared to Win32

Please see the code examples inside mdbglibtest folder for a quick intro.

Foders description:
  1. ASDumpAnalyzer - the project that this library was first intended for. It is used to exctract queries from Microsoft SQL Server Analysis Services crash dumps
  2. dbgengtest - a native project that mimics the functions implemented inside mdbglibtest using direct access to Debugger Engine API. Its reason is to show the difference
  3. Docs - some useful links
  4. dumps - the dumps and applications that were created for unit tests
  5. mdbglib - the actual project. It is written in C++/CLI (managed C++)
  6. mdbglib_cs - an early version of the library that used to reference mdbgeng.dll - the managed wrapper around dbgeng.dll which is not yet publicly available
  7. mdbglibtest - unit tests for mdbglib, examples of different types of operations that can be performed using mdbglib
  8. PowerShellScripts - examples of PowerShell scripts that execute tasks using mdbglib
